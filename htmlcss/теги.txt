<p></p> - параграф, ну типа просто текст лол
<a href ="" ></a> - гиперссылка
<h1>...</h5> - заголовки лол чиуауа маинез
<br> - разделение строки

							СПИСКИ
--------------------------------------------------------------------------------------------------------------------

<ol>
    <li>элемент 1</li>  	\
    <li>элемент 2</li>       	 > Нумерованный список
    <li>элемент 3</li>		/
</ol>

<ul>
    <li>Первый элемент</li>	\
    <li>Второй элемент</li>	 > Ненумерованный список (Маркер просто стоит, ну типа точка)
    <li>Третий элемент</li>	/
</ul>

							ТАБЛИЦЫ
--------------------------------------------------------------------------------------------------------------------

<tr>
        <th>Возраст</th>	\

        <th>Возраст</th>	/
</tr>

<tr>
        <td>Возраст</td>	\
	<td>Имя</td>		 > td обозначает обычную ячейку таблицы с данными.
        <td>Возраст</td>	/
</tr>

							ТЕГ <a>

Атрибут accesskey --> позволяет активировать ссылку с помощью хот кеев, обычно alt + s(ну в яндексе и хроме так)
		Пример: <a accesskey="s" href="https://www.twitch.tv/">ЭЩКЕЕЕЕРЕРЕРЕРЕЕРЕ</a>		
href = --> слово с гиперссылкой
